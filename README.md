# Liquid Glass UI Kit

A modern **Liquid Glass UI Component Library** inspired by glassmorphism and premium interface design.

This project provides clean and elegant UI components with **soft gradients, blur effects, and rounded layouts** suitable for modern web applications.

---

## ✨ Features

- Liquid glass UI design
- Smooth gradient backgrounds
- Glassmorphism blur effects
- Clean and minimal interface
- Reusable UI components
- Modern rounded layouts
- Lightweight and easy to integrate

---

## 📦 Components

The UI kit includes several interface elements such as:

- Primary Action Button
- Icon Action Buttons
- Search Input Fields
- Toolbar Controls
- Toggle Switch
- Tabs
- Modal Components

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
- CSS `backdrop-filter`
- Flexbox / Grid Layout

---

## 📱 Responsive Design

This UI kit is optimized for:

- Desktop
- Tablet
- Mobile
